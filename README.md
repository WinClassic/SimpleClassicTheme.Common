# Simple Classic Theme Common
[![CodeFactor](https://www.codefactor.io/repository/github/winclassic/simpleclassictheme.common/badge)](https://www.codefactor.io/repository/github/winclassic/simpleclassictheme.common)

A library for the Simple Classic Theme suite.

## What does it do?

- **`SimpleClassicTheme.Common.Configuration`**: Shared configurations
- **`SimpleClassicTheme.Common.ErrorHandling`**: Crash handling and error reporting over [Sentry](https://sentry.io/)
- **`SimpleClassicTheme.Common.Logging`**: Simple logger
- **`SimpleClassicTheme.Common.Native`**: Managed implementations of native Windows APIs.
- **`SimpleClassicTheme.Common.Performance`**: Performance logging
- **`SimpleClassicTheme.Common.Serialization`**: Registry wrapper
